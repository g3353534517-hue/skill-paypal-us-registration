# PayPal US Registration Guide / PayPal 美区注册指南

![分类](https://img.shields.io/badge/分类-效率工具-green)

## 项目简介

非美国居民（中国大陆）注册 PayPal 美国账户的完整指南，配合 Wise 美元收款账户使用。涵盖地址策略、SSN 处理、银行绑定和风险评估。

## 功能特性

- PayPal US Registration Guide (for Chinese users with Wise USD account)
- Context
- Recommended Registration Info
- Address Strategy
- Phone
- SSN
- Bank Binding (Wise USD)
- Key Risks (HONEST assessment)
- Recommended Approach
- Alternative Routes if PayPal→Wise Fails
